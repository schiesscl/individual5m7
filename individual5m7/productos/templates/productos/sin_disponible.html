Explicación de defer():

El método defer() le indica a Django que NO cargue el campo especificado en la consulta SQL inicial.

  • ✅ Reduce el tamaño de los datos transferidos desde la base de datos
  • ✅ Mejora el rendimiento cuando no necesitas todos los campos
  • ⚠️ Si intentas acceder al campo excluido, Django hará una consulta adicional automáticamente

Opuesto: only('campo1', 'campo2') carga SOLO los campos especificados.
