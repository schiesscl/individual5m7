No hay productos con precio menor a $50
Query SQL ejecutada:
SELECT * FROM productos_producto WHERE precio < 50
Método usado:
Producto.objects.raw('SELECT * FROM productos_producto WHERE precio < 50')
Explicación:

El método raw() permite ejecutar consultas SQL directamente en Django:

  • ✅ Retorna objetos del modelo Producto
  • ✅ Permite usar SQL nativo cuando el ORM es limitado
  • ✅ Los campos deben coincidir con el modelo
  • ⚠️ El campo id es obligatorio en el SELECT
