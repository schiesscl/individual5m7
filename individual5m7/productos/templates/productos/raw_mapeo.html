Explicación del mapeo:

Los campos seleccionados en el SQL se mapean automáticamente a los atributos del modelo Producto:

  • id → producto.id
  • nombre → producto.nombre
  • precio → producto.precio
  • disponible → producto.disponible

Importante: El campo id es obligatorio en consultas raw().
