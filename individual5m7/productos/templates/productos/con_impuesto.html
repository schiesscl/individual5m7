Anotación usada:
Producto.objects.annotate(
    precio_con_impuesto=ExpressionWrapper(
        F('precio') * Value(Decimal('1.19')),
        output_field=DecimalField(max_digits=10, decimal_places=2)
    )
)
Explicación de annotate():

El método annotate() agrega campos calculados temporalmente a cada objeto del QuerySet:

  • F('precio') - Referencia al campo precio de la base de datos
  • Value(Decimal('1.19')) - Valor decimal exacto para la multiplicación
  • ExpressionWrapper() - Permite operaciones matemáticas complejas
  • output_field - Define el tipo de dato del resultado con 2 decimales

Ventaja: El cálculo se hace en la base de datos, no en Python.

¿Por qué usar annotate()?
  • ✅ El cálculo se realiza en la base de datos (más eficiente)
  • ✅ Permite ordenar o filtrar por el campo calculado
  • ✅ Reduce la carga en el servidor de aplicaciones
  • ✅ El campo calculado está disponible como atributo del objeto
  • ✅ Se mantiene precisión decimal sin errores de redondeo
