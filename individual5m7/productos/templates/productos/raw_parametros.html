No hay productos con precio mayor a ${{ precio_minimo }}
Query SQL con parámetros:
SELECT * FROM productos_producto WHERE precio > %s
Método usado:
Producto.objects.raw('SELECT * FROM productos_producto WHERE precio > %s', [precio_minimo])
Beneficios de usar parámetros:
  • ✅ Previene inyección SQL: Los parámetros se escapan automáticamente
  • ✅ Código más limpio: Separa la lógica de los datos
  • ✅ Reutilizable: La misma query puede usarse con diferentes valores
  • ❌ NUNCA usar: f"...WHERE precio > {valor}" - vulnerable a SQL injection
