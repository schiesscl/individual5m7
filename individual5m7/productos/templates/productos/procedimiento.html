Un procedimiento almacenado es un conjunto de instrucciones SQL que se almacenan en la base de datos y pueden ser invocadas por su nombre. Son como "funciones" en la base de datos.

Crear el procedimiento en MySQL:

CREATE PROCEDURE obtener_productos_por_precio(IN precio_minimo DECIMAL(5,2))
BEGIN
    SELECT * FROM productos_producto WHERE precio >= precio_minimo;
END //
DELIMITER ;
